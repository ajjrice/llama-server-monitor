📺 Chrome Split View Setup

  1. Click "Open Both" above, or open each link in separate tabs
  2. Right-click each tab and select Pin (optional)
  3. Drag one tab out to create a second window
  4. Position windows side by side:
    • Windows: Win + ← and Win + →
    • macOS: Click green fullscreen button → Split View
    • Manual: Resize windows to 50% width each
  5. Control UI on left, Monitor on right (or your preference)

💡 Tip: Once arranged, you can bookmark both URLs and reopen them in your preferred layout anytime.
